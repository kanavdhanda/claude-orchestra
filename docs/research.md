# Research Log

Papers that actually shaped a decision or a piece of architecture in this repo.
Not a reading list — add a paper only once it influences something concrete.

## How to add an entry
1. Create `docs/research/<short-kebab-slug>.md` using the template below.
2. Add one line under Entries: `- [Title](research/<slug>.md) — one-line takeaway`
3. Fill `influences:` with the decision(s)/architecture anchor(s) it affects, and
   drop a matching `[[<slug>]]` wikilink into that decision's/architecture's prose
   if it reads naturally there.

## Template (`docs/research/<slug>.md`)

```
---
tags: [kebab, keywords]
related: []
influences: []
---
# <Paper Title>

**Link:** <arxiv/scholar URL>

## Key Points
- 

## How It's Influencing Our Approach


## What It's Influencing
- [[0000-example-decision]]
```
