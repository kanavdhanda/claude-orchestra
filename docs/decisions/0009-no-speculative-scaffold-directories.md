# 0009. No Speculative Scaffold Directories

**Status:** Accepted

## Context
A set of scaffold directories (`orchestrator/`, `agents/`, `memory/`, `cache/`, `summaries/`, `prompts/`, `tools/`, `configs/`) was originally proposed.

## Decision
Not created. Nothing concrete needs to live in them yet (YAGNI); add if and when a real need appears.

## Consequences
No gap — directories can be created later if a concrete need arises.
