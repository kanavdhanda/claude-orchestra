# 0010. Use Official ast-grep MCP Instead of Building a Custom One

**Status:** Accepted

## Context
A brand-new custom MCP server for ast-grep was considered.

## Decision
Not needed. An official, actively-maintained one already exists (`ast-grep/ast-grep-mcp`, same GitHub org as the CLI) and was registered instead of building a wrapper from scratch.

## Consequences
No gap — the official server covers the need.
