# 0005. Exclude Papers with Code MCP

**Status:** Accepted

## Context
A Papers with Code MCP wrapper was considered for linking papers to implementations.

## Decision
Excluded outright. The underlying paperswithcode.com API/service was shut down by Meta in mid-2025; any wrapper would front a dead service.

## Consequences
No replacement — the capability is simply unavailable now that the upstream service is gone.
