# 0006. No Hard Token Governor

**Status:** Accepted

## Context
A hard "Token Governor" was considered to enforce token budgets programmatically.

## Decision
Not built. No Claude Code hook or API exists to enforce token budgets programmatically. Downgraded to advisory guidance instead.

## Consequences
Token budget discipline is advisory only, not enforced.
