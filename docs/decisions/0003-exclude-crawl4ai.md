# 0003. Exclude Crawl4AI

**Status:** Accepted

## Context
Crawl4AI was considered for web crawling/scraping.

## Decision
Excluded. Firecrawl is already installed and working for crawling/scraping, making Crawl4AI redundant.

## Consequences
None currently. Revisit only if true offline/local crawling becomes a hard requirement, since Firecrawl is a hosted service.
